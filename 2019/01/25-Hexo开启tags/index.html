tags 标签页

  1. 开启标签页

  2. 修改 博客目录/source/tags/index.md 中的数据

  3. 查看 博客目录/scaffolds/post.md 中有无 tags:，若无,则按照格式手动添加并保存

至此完成tags的开启

在博客中显示 tags

  1. 新建博客内容

  2. 打开 博客目录/source/_post/xxxx.md 进行Markdown编辑，在 tags: 后添加自定义标签即可
